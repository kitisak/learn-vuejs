# Step to learning

### Install Vue CLI
```
$npm install -g @vue/cli
# OR
$yarn global add @vue/cli

$vue --version
@vue/cli 4.1.2
```

### Create project with vue CLI
```
$vue create hello
$cd hello
$npm run serve
```
